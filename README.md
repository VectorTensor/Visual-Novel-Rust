## Getting Started

### Prerequisites

- Rust (latest stable version)
- Cargo

### Running the Game

```bash
cargo run
```

### Building for Release

```bash
cargo build --release
```

## Tech Stack

- **Bevy 0.17.3** - Game engine
- **bevy-inspector-egui** - Debug inspector for development

## Development

This template is designed to be a starting point for your visual novel projects. Modify the game logic, add your own assets, and extend the functionality to create your unique story-driven experience.
